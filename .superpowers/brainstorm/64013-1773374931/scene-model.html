Scene Model & Device Geometry

How devices describe their physical shape and position in space

Device Geometry Types

Three geometry types cover all current and planned hardware
PointGeometry
1 LED, single position
LIFX bulbs, single-color Govee
StripGeometry
N LEDs along a direction vector
LIFX strips, OpenRGB LED strips
MatrixGeometry
W×H LED grid with tile offsets
LIFX tile chains (5× 8×8 tiles)

Data Model

New types in types.py
# Geometry types — device reports its physical shape
@dataclass(frozen=True, slots=True)
class PointGeometry: # 1 LED at device position
pass

@dataclass(frozen=True, slots=True)
class StripGeometry:
led_count: int
direction: tuple[float, float, float] # unit vector along strip
length: float # meters

@dataclass(frozen=True, slots=True)
class MatrixGeometry:
tiles: tuple[TileLayout, ...] # per-tile offset + dimensions

@dataclass(frozen=True, slots=True)
class TileLayout:
offset_x: float # relative to device position
offset_y: float
width: int # 8 for LIFX tiles
height: int # 8 for LIFX tiles

# Union type for any geometry
DeviceGeometry = PointGeometry | StripGeometry | MatrixGeometry

# Scene placement — where a device sits in the room
@dataclass(frozen=True, slots=True)
class DevicePlacement:
device_id: str # matches DeviceInfo.name
position: tuple[float, float, float] # (x, y, z) meters
geometry: DeviceGeometry

World-Space LED Positions

Each geometry type expands to per-LED world coordinates
PointGeometry
→ 1 position: device.position
StripGeometry
→ N positions: position + direction * (i / N) * length
MatrixGeometry
→ W×H positions per tile: position + tile.offset + (col, row) * pixel_pitch
Positions computed once at scene load, cached as numpy arrays for fast compositor sampling.

TOML Configuration

Example scene definition in config.toml
# Spatial scene definition
[scene]
mapping = "linear" # default spatial mapping

# LIFX bulb — just a point in space
[[scene.devices]]
name = "living_room_lamp"
position = [2.0, 1.5, 1.2]
geometry = "point"

# LED strip along a shelf
[[scene.devices]]
name = "shelf_strip"
position = [0.0, 3.0, 0.8]
geometry = "strip"
direction = [1.0, 0.0, 0.0] # runs along X axis
length = 1.5 # meters

# LIFX tile chain — tiles auto-discovered from device
[[scene.devices]]
name = "dj_booth_tiles"
position = [1.0, 0.0, 1.5]
geometry = "matrix" # tile layout from LIFX discovery
